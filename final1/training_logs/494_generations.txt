###### ORIGINAL PROMPT #####

MIlle is making snack packs for her kindergarten class. She's going to divide 64 pretzels, four times as many goldfish, and 32 suckers into baggies for the 16 kids in the class. How many items does each baggie have?

#### ANS ####

22

- Pretzels: Since there are 64 pretzels, each baggie will have 64 / 16 = 4 pretzels.
- Goldfish: Since there are four times as many goldfish as pretzels, there are 4 * 64 = 256 goldfish. This makes each baggie have 256 / 16 = 16 goldfish.
- Suckers: There are 32 suckers, so each baggie will have 32 / 16 = 2 suckers.

Now we add up the items in each baggie:

4 pretzels + 16 goldfish + 2 suckers = 22 items per baggie.
